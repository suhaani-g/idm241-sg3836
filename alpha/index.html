IDM241 Alpha – Product Card Hover

glossier.com/collections/fragrance

Triggers

Product Card:

  • Hover On
  • Hover Off

Rules

Product Card:

Hover On

  • The image slightly scales up
  • The image brightness increases
  • The Image fades from bottle to face over .5 seconds
  • Cursor changes to finger

Hover Off

  • The image slightly scales down
  • The image brightness deceases
  • The image fades from face to bottle over .5 seconds

Feedback

  • The image grows slightly, brightens, and gains subtle depth
  • The image transition to a different image
  • The cursor changes to a pointer (hand) to signal clickability
  • Subtext changes from "Eau de parfum" to "50 ml"

Loops & Modes

  • The animation repeats every time you hover on and off a card.
